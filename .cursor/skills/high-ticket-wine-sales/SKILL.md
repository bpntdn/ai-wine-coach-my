# High Ticket Wine Sales

## 使用時機
- 使用者要產出課程銷售文案、社群內容、私訊話術、通話腳本、直播收單稿。
- 使用者目標是提升 35,800 元主課程的預約率與成交率。

## 核心原則
- 先成交再擴散：先優化轉換，再擴大流量。
- 一切輸出以「場景結果」為核心，而不是酒知識堆疊。
- 每份內容都必須有明確 CTA（私訊關鍵字或預約表單）。

## 標準輸出流程
1. 定義受眾（商務 / 社交 / 感情）
2. 套用文案骨架（痛點 -> 代價 -> 解法 -> 案例 -> CTA）
3. 產出平台版本（FB/Threads、IG Reels、YouTube Shorts、YouTube 長片、小紅書、抖音）
4. 提供追單腳本與反對處理句型
5. 附上 KPI 追蹤建議

## 文案模板

### 短影音模板（30-60 秒）
- 鉤子：你以為會喝酒就能拿下關係，其實關鍵是對話節奏。
- 觀點：多數人輸在不會把氣氛導向結果。
- 步驟：先開場破冰，再追問需求，最後提出下一步邀請。
- CTA：私訊「成交」拿完整對話腳本。

### 長文模板
- 開頭：點出常見卡點與損失
- 中段：給出可操作框架
- 結尾：案例 + CTA

## 診斷通話模板
- 開場對焦 -> 現況問題 -> 根因定位 -> 方案邀約 -> 收單確認

## 反對處理模板
- 太貴：回到投資回報與回本週期
- 再想想：拆解真實顧慮（風險/時間/適配）
- 沒時間：強調流程化省時與低試錯成本
- 怕做不到：拆成最小行動與陪跑機制

## 輸出品質檢查
- 是否有明確受眾？
- 是否有具體場景？
- 是否有案例或前後對比？
- 是否有單一明確 CTA？
