# 自動化與資安藍圖（阿科執行版）

## 執行原則
- 先保護你：個資、帳號、支付資訊優先
- 先減少你操作：可自動化的流程盡量系統化
- 先保證成交：所有內容都以高價課預約與成交為目標

## 可立即自動化的部分（我可協助規劃與產出）
- 內容生產標準化：主題庫、腳本模板、平台改寫模板
- 銷售流程標準化：私訊腳本、診斷通話腳本、追單節奏
- KPI 追蹤模板：每日紀錄、每週復盤、A/B 測試框架
- 規則與技能固化：Cursor rule/skill 已建立，可重複產出

## 需要你一次性授權/手動的部分（資安限制）
- 社群平台帳號註冊（FB、IG、YouTube、抖音、小紅書）
- Gmail 建立與兩步驟驗證
- 第三方平台 API 金鑰與權限授權

## 資安最小權限清單
- 所有帳號開啟 2FA（建議 Authenticator）
- 密碼管理器統一管理，不共用密碼
- API 金鑰分環境存放，避免寫進程式碼
- 對外協作使用子帳號，不用主帳號直連

## 低負擔操作設計（你每天只要做這些）
1. 核准當日腳本（10 分鐘）
2. 錄 2-3 支短影音（30-45 分鐘）
3. 回覆高意圖私訊（20 分鐘）
4. 其餘交由既定 SOP 與模板執行
