# 高意圖轉單漏斗與資產清單

## 漏斗總覽

短內容吸引 -> 私訊關鍵字/填表 -> 診斷通話 -> 限時邀約 -> 跟進成交

## 1) 一頁式銷售頁結構（Landing Page）

1. 首屏標題：點出三大結果（成交率、社交力、關係修復）
2. 痛點區塊：列出「會喝但不會用」造成的代價
3. 解法區塊：5 模組課程架構
4. 見證區塊：前後對比與故事
5. FAQ：時間、程度、價格疑慮
6. CTA：預約診斷通話按鈕

## 2) Google 表單欄位建議

- 姓名
- 聯絡方式（Line/IG/Email）
- 目前主要情境（商務 / 社交 / 感情）
- 目前最困擾問題
- 想在多久內看到結果（7 天 / 30 天 / 90 天）
- 是否可接受投資學習（10k、30k、50k+）
- 可預約時段

## 3) Gmail 自動回覆模板

主旨：已收到你的申請｜葡萄酒社交成交力診斷

內文：
您好，已收到你的申請。
我們會先評估你的目標與現況，並安排 15-30 分鐘診斷通話。

通話前請先準備：

1. 最近一個你覺得「有機會但沒拿下」的情境
2. 你目前最想改善的結果（成交/社交/關係）

若你想加速，直接回覆這封信「我想先看案例」，
我會先提供一份可立即使用的情境腳本。

## 4) 預約流程 SOP

1. 收到表單後 12 小時內回覆
2. 先文字確認核心目標與時間窗口
3. 發送預約連結（固定可約時段）
4. 通話前 2 小時提醒
5. 通話後 30 分鐘內發送方案摘要與付款連結

## 5) 追單節奏

- D0：通話後立即發送個人化建議與方案
- D1：補充一個對應案例
- D3：處理剩餘疑慮（時間/價格/信任）
- D5：限時名額提醒（明確截止時間）
