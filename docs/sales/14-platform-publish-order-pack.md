# 平台發布順序與素材對照包

## 優先順序（固定）
1. IG Reels
2. YouTube
3. FB
4. 台抖
5. 陸抖
6. 小紅書
7. bilibili

## 素材規格（統一先做）
- 直式短片：9:16、1080x1920、30-60 秒
- 橫式長片：16:9、1920x1080、2-3 分鐘
- 字幕：繁中白字 + 半透明黑底
- CTA：私訊關鍵字 `成交`

## 每平台直接使用規則

### 1) IG Reels
- 用檔案：Shorts A/B/C（9:16）
- 發布節奏：每週一、三、五
- 文案來源：`12-publish-copies-video01.md` 的 IG 區塊

### 2) YouTube
- 用檔案：長片 16:9 + Shorts 切條
- 發布節奏：每週二長片、週四補 Shorts
- 文案來源：`12-publish-copies-video01.md` 的 YouTube 區塊

### 3) FB
- 用檔案：Shorts A/B（9:16）
- 文字：精簡版（2-4 行）+ CTA

### 4) 台抖
- 用檔案：Shorts A/C（9:16）
- 開場 3 秒保留痛點鉤子

### 5) 陸抖
- 用檔案：Shorts B/C（9:16）
- 口語化標題，避免太長句

### 6) 小紅書
- 用檔案：Shorts B（9:16）+ 條列筆記文案
- 文案：先故事再重點清單

### 7) bilibili
- 用檔案：長片 16:9 為主，Shorts 作導流
- 標題：教學型 + 結果導向關鍵詞
