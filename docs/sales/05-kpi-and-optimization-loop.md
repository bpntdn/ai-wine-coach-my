# KPI 追蹤與每週優化迴圈

## 每週必看指標

- 內容觸及數（平台分開看）
- 私訊率（私訊人數 / 觸及）
- 表單轉換率（填表 / 私訊）
- 預約到場率（到場 / 預約）
- 成交率（成交 / 到場）
- 客單價與總營收

## 每日紀錄格式

- 今日主題：
- 發佈平台與數量：
- 私訊關鍵字數：
- 表單新增數：
- 預約數 / 到場數：
- 成交數 / 金額：
- 今日卡點：
- 明日調整：

## 每週復盤 SOP

1. 找出轉換最低的一段（觸及->私訊、私訊->預約、預約->成交）
2. 只優化一個關鍵槓桿，避免同時改太多變因
3. 保留高轉換素材，淘汰低效內容格式
4. 更新下週 7 天題庫與話術重點

## A/B 測試建議

- 測試 1：不同開場鉤子（問題式 vs 結果式）
- 測試 2：不同 CTA（私訊關鍵字 vs 直接填表）
- 測試 3：不同案例類型（商務 vs 社交 vs 感情）

## 判斷是否加速擴張

若連續兩週達成以下條件，可增加廣告或合作導流：

- 預約到場率 >= 60%
- 成交率 >= 20%
- 內容穩定日更 >= 5 天 / 週
