# 一題多平台內容引流系統

## 每日主題流程
1. 選一個痛點主題（成交 / 社交 / 關係）
2. 先產 60 秒短影音腳本
3. 再延伸成長文與長影片
4. 全平台統一 CTA（私訊關鍵字或填表）

Day7：案例拆解（前後對比）

## 平台改寫模板

### FB / Threads 貼文
- 結構：痛點開場 -> 案例 -> 做法 -> CTA
- 長度：200-500 字

### IG Reels / YouTube Shorts / 抖音
- 結構：3 秒鉤子 -> 一個觀點 -> 一個可行動步驟 -> CTA
- 長度：30-60 秒

### YouTube 長影片
- 結構：問題背景 -> 框架教學 -> 真實案例 -> 作業 -> CTA
- 長度：8-12 分鐘

### 小紅書
- 結構：情境故事 -> 條列重點 -> 可抄作業句 -> CTA
- 形式：封面標題 + 清單筆記

## 直播節奏（每週一次）
- 前 10 分鐘：破題與常見錯誤
- 中間 35 分鐘：教學 + 案例
- 後 15 分鐘：Q&A + 限時收單
